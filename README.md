## Mob Bridge

### Description

Mob registration compatible with `mobkit` & `mobs_redo`.

***WARNING:** this mod is in early development & not currently functional*

### Usage

Methods:
```lua
mobb.register(def)
- Registers a new mob with enabled library.
- `def`: entity definition
```
